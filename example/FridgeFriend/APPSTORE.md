# FridgeFriend — App Store Listing

## App Name (25/30 characters)

```
FridgeFriend: Food Expiry
```

Brand first, then the two highest-intent search terms the brand doesn't already cover. "Fridge" is inside the brand name and still indexes.

## Subtitle (27/30 characters)

```
Track pantry & fridge dates
```

Zero overlap with the name's keywords ("track", "pantry", "dates" are all new index terms), and it reads as a plain-language promise, not a keyword pile.

## Keyword Field (99/100 characters)

```
expiration,use by,best before,waste,grocery,kitchen,freezer,leftovers,shelf life,reminder,organizer
```

Rules applied: no words already in the name or subtitle (Apple indexes those for free), no spaces after commas, singulars only (Apple stems plurals), "expiration" covers the US spelling the name's "expiry" misses.

## Promotional Text (97/170 characters)

```
Track every expiry date in your kitchen. Add an item in five seconds, see what to use up tonight.
```

## Description

*(First three lines are the entire pitch — they're all most people see above "more".)*

```
Stop paying for food you throw away.
FridgeFriend tracks what's in your fridge, freezer, and pantry — and shows you what to use up before it turns.
No account. No subscription. No camera pointed at your groceries. Just a list that knows the dates.

HOW IT WORKS
• Add an item in seconds: name, date, done
• Everything sorts itself: Expiring Soon, Fresh, Expired
• Color badges show exactly how long each item has — "Tomorrow", "5d left", "2d ago"
• "Expires today" still counts as usable — the app tells you to cook it, not toss it

BUILT FOR REAL KITCHENS
• Fridge, freezer, and pantry, each with its own icon
• Quantities for multiples, notes for things like "opened Tuesday"
• Log the mystery jar from last month — past dates are allowed on purpose
• Edit or delete anything in two taps

SEE THE BIG PICTURE
• Stats show what's fresh, what's about to turn, and what's already gone
• "Use Next" tells you the single item to reach for first

PRIVATE BY DESIGN
• Works fully offline, forever
• No account, no ads, no tracking, no data collected — the privacy label is empty because there's nothing to put on it

FridgeFriend does one job: it remembers the dates so you don't have to.
```
